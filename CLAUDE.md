# claude-config-helper

- Claude config **review** tasks → `claude-config-reviewer` subagent
- Claude config **generation/scaffolding** tasks → `claude-config-generator` subagent
- Claude config **full setup** (generate + review) → `setup-claude-config` skill
- Review criteria are in the `review-claude-config` skill
- Generation templates are in the `generate-claude-config` skill
